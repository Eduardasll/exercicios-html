# Exercícios de HTML

Repositório com exercícios práticos desenvolvidos durante o curso "The Complete Full-Stack Web Development Bootcamp" (Udemy), focando na construção da base da linguagem HTML.

## Conteúdos praticados:
- Estrutura básica
- Títulos e parágrafos
- Listas
- Links
- Imagens

Objetivo: consolidar fundamentos antes de avançar para projetos mais complexos.
